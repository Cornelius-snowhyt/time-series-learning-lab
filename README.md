# Time Series Learning Lab

This project explores fundamental time series analysis techniques using Python.

## Objectives
- Understand time series components
- Apply statistical modeling
- Implement forecasting techniques

## Techniques Used
- Linear Regression
- Seasonal Decomposition
- Rolling Averages
- ARIMA Forecasting
- SARIMA Forecasting
- Residual Autocorrelation Analysis

## Dataset
Airline passenger dataset containing monthly passenger totals.

## Libraries
- pandas
- numpy
- matplotlib
- statsmodels
- scikit-learn

## Key Visualizations
- Electricity usage vs temperature regression
- Airline passenger trend analysis
- Seasonal decomposition
- Rolling averages
- ARIMA forecasting
- SARIMA forecasting with confidence intervals

## Learning Outcomes
This project demonstrates:
- Time series trend detection
- Seasonal pattern analysis
- Forecasting using statistical models
- Residual diagnostics using ACF
